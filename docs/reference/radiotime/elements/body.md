# body

Standard OPML container element wrapping a response payload — one
or more `<outline>` elements. The original spec page for `<body>`
was not preserved in the Wayback archive.

## `<body>` element

No archived spec page available — the Wayback snapshot for this URL is a
404 "Page not found" placeholder rather than the original developer
documentation.  The `<body>` element is referenced by the API matrix and
by sibling pages (`<head>`, `<opml>`, `<outline>`) but its own page was
not preserved.  Treat `<body>` as the standard OPML container element
that wraps the response payload (one or more `<outline>` elements).
